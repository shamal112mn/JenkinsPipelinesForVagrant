Jenkins Pipeline Folders for Vagrant Projects

```
Look in the folder for Jenkinsfile

Dockerhub-pushImage -> pipeline to build docker image and push to deockerhub

DockerPepilines -> pipeline how to use docker container on agent

JenkinsJobSourceCodes -> examples in text of jenkins pipelines

PipelineWithGroovy -> simple pipeline example with groovy

Simple-Webpage -> simple pipeline to deploy index.html page on Nginx 

```
